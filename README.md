Основные юредические лица Сбербанка: ПАО Сбербанк,ООО СК «Сбербанк страхование», ООО СК «Сбербанк страхование жизни», ООО «Сбербанк Капитал», ООО «Сбербанк Факторинг»,ООО «ДубльГИС» (2ГИС)\
Площадки для поиска информации:\
ЕИС — обязательный фундамент для работы с госзаказом. Однако для эффективного поиска и аналитики требует дополнения агрегаторами.\
Сбербанк-АСТ — крупнейшая площадка по объему госзакупок и ключевая для работы с дочерними обществами крупных корпораций. Участие в тендерах на этой площадке критически важно для большинства направлений бизнеса.\
Дополнительные источники:\
РосТендер — инструмент для расширенного поиска, особенно если бизнес работает в строительстве или с корпоративными заказчиками, чьи закупки не всегда полностью отражаются в ЕИС.\
--Между ЕИС и другими площадками возможны расхождения в ценах/датах из-за задержек синхронизации\
--Данные источники были выбраны по двум причинам: их функционал обеспечивает максимальное удобство при работе с информацией, а объём и полнота представленных в них сведений полностью покрывают потребности для подготовки сводки.\
